# The Final Crawl

This is a project made for my Computer and Information Science class. This is no wheres near well made game.

## How to run the game

### Windows
To run the game on windows you can either double click the main.py file to open it or you can go to the IDLE application and open it in there.

### MacOS
You can either use the terminal and use the cd command to navigate to the file than run it with python3 main.py

or

You can open in it the IDL application

### Linux
The best way to run it is in the terminal. Just like MacOS you can use cd to get to the file than do python3 main.py


### Other Choices
On all 3 you can run it through Visual Studio Code
